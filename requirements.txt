## Como Usar

1.  **Clone o repositório (ou crie os arquivos manualmente):**
    ```bash
    git clone <URL_DO_SEU_REPOSITORIO>
    cd flashcard_generator
    ```

2.  **Crie e ative um ambiente virtual (recomendado):**
    ```bash
    python -m venv venv
    # No Windows
    .\venv\Scripts\activate
    # No macOS/Linux
    source venv/bin/activate
    ```

3.  **Instale as dependências:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Execute a aplicação Streamlit:**
    ```bash
    streamlit run app.py
    ```

    Isso abrirá a aplicação no seu navegador padrão (geralmente em `http://localhost:8501`).

## Funcionalidades

* Definição de Assunto Principal e Matéria.
* Entrada de tópicos detalhados (com suporte a sub-tópicos).
* Opções configuráveis para o formato de saída do flashcard, nível de detalhe, estilo da pergunta e foco do conteúdo.
* Opções para incluir erros comuns, tags, grau de dificuldade e contexto do ENEM.
* Geração e exibição do prompt completo.
* Botão para copiar o prompt gerado.
